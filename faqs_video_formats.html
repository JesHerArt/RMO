Frequently Asked Questions

Q

What is the difference between video/audio formats?

A

On our site we have 5 different formats that we use interexchangbly and 4 formats are available for your to download. Please see descriptions below:

  • .Mp4
    • Mp4 is a container format and its abbreviation stands for MPEG-4. This type of container format allows you to add a combination of video, audio, subtitles and still images to it’s one single file. The Mp4 video file has been around since 1998 and to this day it’s still one of the most used video file formats to use on the web. Since this could be considered a legacy video format, Mp4 is still able to be used with older versions of internet browsers (i.e. Internet Explorer). By using a flash fallback, the Mp4 file can be used inside of flash to therefore display the video on an older browser. Versatility is one of the bonuses you get by using a Mp4 file type.
  • .Ogv
    • Ogv is an open sourced bit stream container. This file type was created by the Xiph.org Foundation. Ogv’s have two components to them within their codecs. The Theora codec handles the video portion or an Ogv file, while the Vorbis codec handles the audio portion. Aside from video and audio, it can also support subtitles. In a whole, this is a lossy video compression format and was created for streaming and manipulation. Ogv file formats are typically used in the Firefox or Opera web browsers, but it is also capable of being used on the Google Chrome browser as well.
  • .WebM
    • WebM began as an open web media project and it is entirely royalty free format to use by anyone in combination with the HTML5 video tag. WebM is a fairly new video format that was created in 2010 and based off of the Matroska media container. It consists of a VP8 or VP9 video codec and a Vorbis or Opus audio format. WebM can also support WebVTT text titles. Avoiding patent issues is one of the reasons why WebM was created. This format is supported by the newer versions of the top browsers (i.e. Chrome and Firefox). WebM is also very useful for the Android, iOS and Windows Mobile web browsers.
  • .Mp3
    • Mp3 format is a pure audio format. Mp3 is the short name for either MPEG-1 or MPEG-2 Audio Layer III. This format was created in 1993 and it’s a form of lossy data compression. Therefore, the audio on a CD will sound better because it’s an original but the Mp3 is used in order to compress the data and would can notice some subtle differences between the original and the Mp3 format. To this day Mp3 is still once of the most popular digital audio formats. It runs on the MPEG audio codec and it used for streaming or storing audio in general.
  • .SWF
    • SWF, also known as a Small Web Format or ShockWave Flash, has been passed down through various companies. SWF was first created by FutureWave Software and it was designed to be able to handle animations, vector graphics, applets, web games and multimedia in general. Later on Macromedia acquired the format and much after that, was acquired yet again by Adobe. SWF is essentially a flash file. Using flash content was an alternative to be able to load multimedia content onto an older browser or slower connections. To this day we are still using flash fallbacks for those who don’t not have the most updated current browser but it’s beginning to get phased out on the newer browsers due to recent discoveries of security risks on the user’s end.
